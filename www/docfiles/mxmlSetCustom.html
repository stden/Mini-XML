mxmlSetCustom()

Description

Set the data and destructor of a custom data node. The node is not changed if it is not a custom node.

Syntax

int
mxmlSetCustom(
    mxml_node_t * node,
    void * data,
    void (*destroy)(void *));

Arguments

NameDescription
nodeNode to set
dataNew data pointer
(*destroy)(void *)New destructor function

Returns

0 on success, -1 on failure
